# Stage 03 — Add-in UX  ◀ current stage

**Goal:** wire the verified analytics into the Excel add-in so the desk can run them from a
ribbon button and see results in a task pane — turning the Python surface into a usable tool.

**Versions:** v0.4.0 (ribbon + task pane calling the analytics), v0.5.0 (error handling +
user-facing diagnostics).

**Milestone (stage exit):** a user can select inputs in Excel, run an analytic from the
ribbon, and see results in the task pane — with failures surfaced clearly, not as crashes.
