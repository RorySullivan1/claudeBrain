# version: v0.5.0
status: planned

## Goals
- Surface analytics/runtime failures to the user as clear, actionable messages.
- Add lightweight diagnostics (a log + a "copy diagnostics" affordance) for support.

## Objectives / acceptance
- A failed analytic shows a meaningful task-pane message, never an unhandled crash dialog.
- Diagnostics capture enough context (inputs, error, version) to triage a desk report.
